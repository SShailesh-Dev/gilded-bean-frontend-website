# The Gilded Bean ☕

A premium, heritage-inspired cafe website built with React, Tailwind CSS, and Framer Motion. **The Gilded Bean** offers a seamless user experience for browsing artisanal brews, booking tables, and ordering online.

## ✨ Features

- **Artisanal Menu**: Browse a curated selection of coffees, beverages, snacks, and desserts.
- **Smart Search & Filtering**: Easily find your favorite items with real-time search and category-based filtering.
- **Interactive Cart**: A smooth, side-drawer cart system to manage your orders.
- **Multi-step Checkout**: A secure and intuitive checkout process with contact, shipping, and payment validation.
- **Table Booking**: Reserve your spot at our heritage cafe through an integrated booking form.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **Smooth Animations**: Engaging entrance and scroll-triggered animations powered by Framer Motion.
- **Toast Notifications**: Real-time feedback for user actions like adding items to the cart or placing orders.
- **Scroll Management**: Automatic scroll-to-top on page navigation for a clean browsing experience.

## 🚀 Tech Stack

- **Frontend**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SShailesh-Dev/gilded-bean-frontend-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd gilded-bean-frontend-website
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### Building for Production

To create a production-ready build:
```bash
npm run build
```
The output will be in the `dist/` directory.

